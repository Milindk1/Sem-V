Various Boxes Supported in JavaScript

JavaScript provides different types of boxes or containers that you can use to store and manipulate data. These boxes include:

1. Alert Box

The alert box is a simple dialog box that displays a message to the user.

2. Confirm Box

The confirm box displays a message and allows the user to choose between "OK" and "Cancel".

3. Prompt Box

The prompt box displays a message and allows the user to enter some text.
